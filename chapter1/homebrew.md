#### Homebrew

##### 安装

```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/instd
```

##### 命令

```bash
brew install #安装软件
brew search #搜索软件
brew uninstall #卸载软件
brew update #更新所有软件
brew upgrade #更新具体软件
brew list #显示已安装软件
brew info #查看软件信息
brew reps #显示包依赖
brew services list #显示安装的服务
brew services start/stop/restart serverName #安装服务启动、停止、重启
```
